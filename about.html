Why make a website?

Websites offer more versatility than posters. While most posters just display information, users can actually interact with the website to learn more and gain a deeper understanding about certain topics. Also, you can put a poster on a website, but not a website on a poster.


Why not use a website template but "hard code"?

Website templates or software that produces websites for you only give you limited options for layout. You can pay premium to expand layout selection, but there are not infinite layouts that you can choose from. Also, if you use a website-builder, there are only so many things you can do. Hard coding a website gives you 100% control over what you want, and you can put, literally, anything on a website that you self-make. It definitely is more time consuming, but it is also extremely fun!
